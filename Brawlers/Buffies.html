📖 O que são os Buffies?

Buffie
🦊
TipoUpgrade Permanente
AfetaGadgets, Star Powers, HC
AtivaçãoAutomática
Custo base1.000 Coins + 2.000 PP
Requisito1.000 Trofeus
1.º PullGRÁTIS

Os Buffies são upgrades permanentes introduzidos no Update 65 (Dezembro 2025) que melhoram as capacidades dos Gadgets, Star Powers e Hypercharge dos Brawlers. São representados visualmente como pequenas porta-chaves que o Brawler carrega consigo.

Ao contrário dos Gears, os Buffies não precisam de ser equipados manualmente — uma vez desbloqueados ficam ativos permanentemente. Cada Brawler tem 3 Buffies disponíveis e podes tê-los todos ativos ao mesmo tempo.

Os Buffies substituíram os Gears Epic e Mythic dos Brawlers que os receberam — esses Gears foram removidos e os jogadores receberam reembolso total em Coins.

💡
O primeiro pull de cada Claw Machine é sempre GRÁTIS. Aproveita sempre o pull gratuito antes de gastar Coins e Power Points.

🎨 Tipos de Buffie

Existem 4 tipos de Buffies, cada um com uma cor de porta-chave diferente. Os 3 primeiros têm efeitos de gameplay; o quarto é puramente cosmético.

🟢
Gadget Buffie
Melhora ambos os Gadgets do Brawler. Pode adicionar um segundo efeito, tornar o Gadget apontável com joystick, ou amplificar o efeito existente.
Porta-chave Verde
🟡
Star Power Buffie
Melhora ambas as Star Powers do Brawler. Geralmente melhora a fraqueza da Star Power ou adiciona um efeito extra complementar.
Porta-chave Amarela
🟣
Hyper Buffie
Melhora o Hypercharge. Todos os Hyper Buffies adicionam +2 segundos de duração ao Hypercharge e aumentam a eficácia do ataque base durante o mesmo.
Porta-chave Roxa
🩷
Bling Buffie
Cosmético exclusivo — não tem efeito de gameplay. Só disponível após desbloquear os 18 Buffies de uma Claw Machine. Custa 10.000 Bling por pull.
Porta-chave Rosa (Cosmético)
ℹ️
Todos os Hyper Buffies partilham um efeito base universal: +2 segundos de duração do Hypercharge e aumento da eficácia do ataque base durante o HC. O efeito específico de cada um varia por Brawler.

💰 Como Obter Buffies

Existem várias formas de desbloquear Buffies. O sistema de Buffies fica disponível para um Brawler quando este atinge 1.000 Trofeus.

🎰
Claw Machine
A forma principal. Cada pull é aleatório mas garante 1 Buffie. O primeiro pull de cada máquina é sempre grátis.
1.000 Coins + 2.000 PP
🔑
Buffie Key (Vault)
Usa uma Buffie Key no Brawl Pass Vault para escolher o Buffie exato que queres — sem aleatório.
1 Buffie Key
💎
Gems
Podes comprar Buffies específicos diretamente com Gems na loja dedicada.
Varia por raridade
💥
Ultra Chaos Drop
Há cerca de 15% de chance de obter um token de Buffie a partir de um Chaos Drop Mythic ou Legendary.
~15% chance (Mythic/Leg.)
🎯
Mega Quest Mensal
A partir do Update 66, cada temporada tem uma Mega Quest que dá um Buffie aleatório como recompensa.
Grátis (completar quest)
🩷
Bling Buffie (Cosmético)
Depois de teres os 18 Buffies de uma Claw Machine, podes puxar Bling Buffies cosméticos.
10.000 Bling
⚠️
Na Claw Machine não podes escolher qual Buffie recebes — é aleatório. Se queres um Buffie específico, usa uma Buffie Key no Vault ou compra com Gems.
